How I Use Raycast

By CJ Dunteman

Hyperkey Shortcuts

I am using Karabiner Elements to map SHIFT + COMMAND + CTRL + OPTION to CAPS LOCK.

Shortcut Action
HYPER + F Open Finder
HYPER + A Open Arc Browser
HYPER + C Open Visual Studio Code
HYPER + S Open Terminal
HYPER + D Open Figma
HYPER + P Open 1Password
HYPER + O Open Outlook
HYPER + T Open Microsoft Teams
HYPER + L Lock screen

Raycast Notes

I use this every day. It is wicked fast, uses Markdown, and is just one shortcut away (OPTION + N). The release of Raycast Notes 2.0 was what finally pushed me to purchase Raycast Pro. The bottom line is that Raycast Notes is simple, effective, and efficient.

AI Chat

Originally I purchased Raycast Pro for the new notes update, but the AI chat has turned out invaluable. So far I have found Claude 3.5 Haiku the best at giving me programming related help, which has been my most crucial usecase.
